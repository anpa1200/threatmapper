# Demo Videos

This directory contains short reviewer-facing workflow videos for AdversaryGraph.

## DFIR Report to AI Analysis to Compare

- Video: [`dfir-report-ai-analysis-compare.mp4`](dfir-report-ai-analysis-compare.mp4)
- GIF: [`dfir-report-ai-analysis-compare.gif`](dfir-report-ai-analysis-compare.gif)
- Poster: [`dfir-report-ai-analysis-compare-poster.png`](dfir-report-ai-analysis-compare-poster.png)
- Duration: 1 minute 51 seconds
- Formats: H.264 MP4, 1600x842, no audio; optimized GIF, 960x505, 8 fps

The demo shows a local analyst workflow:

1. Open indexed DFIR Report examples.
2. Create or select a local PDF from the original source report.
3. Upload the PDF to AdversaryGraph AI Analysis.
4. Stream ATT&CK technique extraction.
5. Review selected TTPs in Navigator for comparison and follow-on analysis.

The video is a compressed derivative of a local desktop screencast and is intended for maintainers, editors, and reviewers. It does not embed or mirror third-party report content; it demonstrates user-driven local analysis from the original source page.
