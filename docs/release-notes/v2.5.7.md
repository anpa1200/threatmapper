# AdversaryGraph v2.5.7 Release Notes

Release date: 2026-06-19

AdversaryGraph v2.5.7 adds MiniMax as a first-class external LLM provider for
AI-assisted CTI analysis workflows.

## Changes

- Added a MiniMax LLM adapter using the official OpenAI-compatible Chat
  Completions API.
- Added deployment settings:
  - `MINIMAX_API_KEY`
  - `MINIMAX_MODEL`
  - `MINIMAX_BASE_URL`
- Added MiniMax to AI Analysis provider selection.
- Added MiniMax to the Navigator AI chat provider selector.
- Added MiniMax to IOC Library and Feeds Management AI-enrichment provider
  dropdowns.
- Added backend provider validation for MiniMax across report analysis and IOC
  enrichment APIs.
- Added MiniMax API-key visibility to `/api/system/selftest`.
- Added Docker Compose environment forwarding for API and worker containers.
- Updated README, quickstart, admin guide, full guide, publication drafts,
  public docs, and the 1200km article mirror.
- Added focused provider factory test coverage.

## Upgrade

```bash
git pull
docker compose up -d --build
```

To use MiniMax, add the following to `.env`:

```env
MINIMAX_API_KEY=your_key_here
MINIMAX_MODEL=MiniMax-M3
MINIMAX_BASE_URL=https://api.minimax.io/v1
```

Then restart the API container:

```bash
docker compose up -d --build api worker frontend
```
