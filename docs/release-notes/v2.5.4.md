# AdversaryGraph v2.5.4 Release Notes

Release date: 2026-06-19

AdversaryGraph v2.5.4 is an IOC normalization and IOC-to-TTP mapping quality
release.

## Changes

- Normalized provider-specific hash IOC types into stable library types:
  `sha256`, `sha1`, and `md5`.
- Added duplicate IOC consolidation when legacy hash labels conflict with an
  already-normalized IOC row.
- Preserved merged IOC metadata, tags, confidence, first/last seen values, raw
  enrichment context, and actor links during duplicate consolidation.
- Added source-priority IOC-to-TTP mapping: strict source/report evidence first,
  enrichment-platform evidence second, and optional AI fallback last.
- Added `/api/ioc/enrich/ttps` for reprocessing local IOC-to-TTP mappings and
  normalizing stored IOC types.
- Added `ai_enrich` and `ai_provider` query options to IOC sync endpoints.
- Added explicit AI-fallback controls in IOC Library and Feeds Management.
- Added focused unit tests for hash normalization and IOC-to-TTP evidence
  priority.
- Updated README, admin guide, and full guide with the IOC mapping policy.

## Upgrade

```bash
git pull
docker compose up -d --build
```

Optional deterministic local IOC reprocessing:

```bash
curl -X POST 'http://localhost:8000/api/ioc/enrich/ttps?limit=20000'
```

To allow AI fallback during sync, enable the checkbox in IOC Library or Feeds
Management, or pass:

```text
ai_enrich=true&ai_provider=local
```
