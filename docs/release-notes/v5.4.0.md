# AdversaryGraph v5.4.0 Release Notes

Release date: 2026-06-02

## Highlights

- OpenCTI symmetric sync: AdversaryGraph can now push completed analysis sessions as STIX 2.1 bundles to a connected OpenCTI instance and pull actor/campaign data back.
- STIX export endpoint (`/api/export/analysis/{id}/stix`) returns a valid STIX 2.1 bundle for any stored analysis session.
- MalwareGraph integration hardened: upload timeout, retry logic, and graceful degradation when MalwareGraph is unavailable.
- Statistics dashboard extended with sector-breakdown and CVE-risk widgets.

## Upgrade Notes

- Set `OPENCTI_URL` and `OPENCTI_TOKEN` to enable OpenCTI sync; leave blank to disable.
- `OPENCTI_VERIFY_TLS` defaults to `true`; set to `false` only for self-signed certs in isolated lab environments.
