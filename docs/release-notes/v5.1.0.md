# AdversaryGraph v5.1.0 Release Notes

Release date: 2026-04-15

## Highlights

- Extended ATT&CK domain coverage to include ATLAS (AI threat landscape) alongside enterprise, mobile, and ICS matrices.
- Added domain-aware technique filtering so multi-domain layers remain consistent with the selected scope.
- Improved session comparison view with side-by-side TTP diff and Jaccard similarity delta.
- Expanded IOC intelligence pipeline: OTX actor-pulse sync now includes malware-family attribution from Malpedia.

## Upgrade Notes

- No breaking schema changes. Existing analysis sessions and layers are compatible.
- ATTCK_DOMAINS now defaults to `enterprise-attack,mobile-attack,ics-attack,atlas`. Override if a narrower set is required.
