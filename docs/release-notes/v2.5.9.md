# AdversaryGraph v2.5.9 Release Notes

Release date: 2026-06-19

AdversaryGraph v2.5.9 expands Detection Studio with a default public YARA rule
source, YARA-L generation, and optional AI-assisted detection rule authoring.

## Changes

- Added the public Yara-Rules malware repository as a default YARA feed source.
- Added explicit raw YARA rule URLs so the Docker runtime does not depend on
  GitHub API tree listing or `git`.
- Added YARA-L detection skeleton generation.
- Added YARA-L structural validation.
- Added optional AI-assisted rule generation for Sigma, YARA, YARA-L, KQL, SPL,
  and EQL.
- Added AI provider selection for detection generation: local, Claude, OpenAI,
  Gemini, and MiniMax.
- Added optional model override, telemetry/event input, and analyst context for
  AI-generated detection rules.
- Stored generation metadata (`generation`, `provider`, `model`) in detection
  validation details.
- Updated Pipeline and Feeds Management UI text.
- Updated README, admin guide, full guide, user guide, release notes, and
  release summary.

git pull
docker compose up -d --build
