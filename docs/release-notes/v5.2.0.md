# AdversaryGraph v5.2.0 Release Notes

Release date: 2026-04-28

## Highlights

- Introduced RetroHunt: a persistent signal store that aggregates NVD CVEs, CISA KEV, and GitHub security advisories for continuous retrospective hunting.
- Added `/api/retrohunt/signals` and `/api/retrohunt/stats` endpoints with full-text and tag-based filtering.
- CVE Library now links CVE records to ATT&CK techniques via evidence-backed relationships (CWE → technique mapping).
- Celery background task queue added for long-running sync and collection operations.

## Upgrade Notes

- Redis is now required. Set `REDIS_URL` in your `.env`.
- Run `alembic upgrade head` to create the `retrohunt_signals` table.
