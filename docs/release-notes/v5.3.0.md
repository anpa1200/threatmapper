# AdversaryGraph v5.3.0 Release Notes

Release date: 2026-05-12

## Highlights

- Added SIEM forwarding: analysis results and audit events can be forwarded to any syslog-compatible SIEM over UDP/TCP.
- Introduced Evidence Graph view: a force-directed graph linking IOCs, techniques, actor groups, and CVEs extracted from a single analysis session.
- Sector tagging for APT groups and CVEs enables sector-filtered dashboard widgets and retrohunt queries.
- Log and PCAP analysis (`/api/analyze/log-pcap`) promoted from beta: now supports `.pcap`, `.pcapng`, `.cap`, and structured log formats.

## Upgrade Notes

- No schema changes. Existing deployments upgrade in place.
- SIEM forwarding is disabled by default; configure `SIEM_HOST` and `SIEM_PORT` to enable.
