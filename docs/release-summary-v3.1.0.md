# AdversaryGraph v3.1.0 Release Summary

AdversaryGraph v3.1.0 packages the published From Log to Report workflow into
the official documentation and 1200km ecosystem.

## Release Focus

- From firewall and EDR logs to a structured investigation report.
- Source-tagged IOC and TTP evidence.
- IOC Investigation pivots and relationship graph review.
- ATT&CK matrix handoff, actor-overlap comparison, AI summary, and report
  generation.
- Full visual documentation using screenshots, infographics, and animated GIFs
  from the published article.
